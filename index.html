Open your Google Sheet → click Extensions in the menu bar → Apps Script. A new tab opens with a code editor.
2
Select all the default code and delete it. Paste in the Apps Script code from the README, then click Save (💾 or Ctrl+S).
3
Click Deploy → New deployment. Click the ⚙️ gear next to "Select type" and choose Web app.
4
Set Execute as: Me and Who has access: Anyone. Click Deploy.
5
Google will ask you to authorize — click Review permissions → Advanced → Go to WeightLog (unsafe) → Allow. (The "unsafe" warning is just because the app isn't published in the Workspace Marketplace — it's your own script.)
6
Copy the Web app URL — it starts with https://script.google.com/macros/s/ — and paste it below.
